  • Mod Commands: (!giv can be changed)
  • !giv start
    Start a 5-min giveaway: !giv start 5m Super Awesome Prize
    Start a 30-sec giveaway: !giv start 30s Quick Everyone Enter!
    Start a No-Time giveaway: !giv start Long Giveaway
  • !giv startredeem
    Start a 5-min channnel point giveaway with 500 cost: !giv startredeem 500 5m Super Awesome Prize
    Start a No-Time channnel point giveaway with 500 cost: !giv startredeem 500 Long Giveaway
  • !giv pick
    Manually Pick a Winner
  • !giv time
    Change the time limit to 1-min: !giv time 1m
  • !giv pause / !giv resume
    Pause or Resume the giveaway timer
  • !giv reset
    Reset and hide the giveaway
  • !giv reopen
    Reopen the giveaway and let new people join without starting over
  • !giv ignore / !giv unignore
    Ignore or Unignore a viewer from entering the current giveaway
  • !giv ban / !giv unban
    Ban or Unban a viewer from entering any giveaway until the tool is restarted
  • Viewer Commands: (commands can be customized)
  • !join / !unjoin
    Join or unjoin a giveaway as a viewer
  • !win
    Claim the prize as the selected giveaway winner
  • !skipme
    Skip the prize as the selected giveaway winner
